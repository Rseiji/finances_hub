# Data Sources

This document lists candidate free data sources. Final selection is **to be evaluated** and should balance rate limits, data quality, and licensing. We will iterate together to identify the best free endpoints for this application.

## Stocks (Candidates)
- Stooq (historical and daily data)
- Alpha Vantage (free tier)
- IEX Cloud (limited free tier)
- Yahoo Finance (unofficial endpoints; verify reliability)
- Twelve Data (free tier)

## Crypto (Candidates)
- CoinGecko (public API)
- CoinCap (public API)
- CryptoCompare (free tier)
- CoinPaprika (public API)
- Exchange APIs (e.g., Binance public endpoints)

## Trades/Transactions (Candidates)
- CSV imports from broker or exchange exports
- Exchange REST APIs where available
- Brokerage exports (PDF/CSV) where permitted

## Selection Criteria
- Free tier and clear usage limits
- Time-series data availability
- Stable API and documentation
- Acceptable licensing for personal use
- Reliability and rate-limit consistency for local scheduling
